# n8n Automation AI

A collection of reusable n8n automation templates powered by AI and cloud integrations

This repository contains a growing collection of n8n workflow templates designed to automate real world tasks using AI, cloud services, and integrations. The templates in this repo are built to help streamline operations, reduce manual work, and provide reusable automation patterns that can be adapted for different projects and environments.

Each workflow is provided as an importable JSON file along with documentation, screenshots, and usage notes to help you deploy and customize it easily.

##What this repository includes
- Ready to use n8n automation templates
- AI assisted reporting and summarization workflows
- Data processing and integration pipelines
- Cloud and cost monitoring automations
- Examples of practical workflow design patterns

## Current Templates
- Azure Cost Monitoring and AI Summary
Automates cost export retrieval, deduplication, Google Sheets updates, and AI generated spending summaries delivered through email.
More templates will be added over time as the collection grows.

## Objectives of this project

- Provide reusable automation templates for n8n users
- Demonstrate practical AI assisted workflow applications
- Encourage learning, experimentation, and workflow customization
- Help teams build repeatable and scalable automation processes

## Technologies used across templates
- n8n workflow automation
- OpenAI API for AI summarization and analysis
- Google Sheets and cloud storage integrations
- JavaScript function logic for workflow control

## How to use the templates
- Import the JSON workflow file into your n8n instance
- Configure credentials and environment variables as required
- Modify nodes or logic to match your environment
- Activate the workflow and test
